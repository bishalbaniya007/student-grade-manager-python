# Student Grade Manager (Python)
A menu-driven Python application that manages student records
using file handling and calculates GPA, grades, class averages,
and highest/lowest scorers.

## Features
- Add student
- View all students
- View specific student
- GPA & Grade calculation
- Highest and lowest scorer
- Class average

## Technologies
- Python
- File Handling
- Dictionaries
- Functions

## Run
python main.py
